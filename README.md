# TimTim Homepage

A free, static Vite landing page for TimTim built with HTML, CSS, and vanilla JavaScript.

## Local commands

```bash
npm install
npm run dev
npm run build
```

## Cloudflare Pages settings

- Framework preset: Vite
- Build command: `npm run build`
- Build output directory: `dist`

This site is static only. It does not require a backend, database, login system, or paid service.
